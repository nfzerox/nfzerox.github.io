A small Cydia / APT repository for legacy iOS.

On an older iOS? Install modern root certificates first. This page and the repo are served over HTTPS, which old iOS can't validate until today's root certificates are on the device (the same reason old Safari can't open modern sites). Open tlsroot.litten.ca in Safari and install the "iOS Bundle", then come back and add the source below. Already stuck and can't load anything? Download the .deb directly and install it yourself instead.

TLSFix

Modern HTTPS for legacy iPhone, iPad, and iPod touch running iOS 3 to iOS 9. Routes the device's HTTPS through a bundled mbedTLS (TLS 1.2 / 1.3, modern ciphers and certificates) so old Safari and apps can reach today's servers again, with full certificate verification.
